# brain-curls

An interactive brain training app scaffold focused on a small set of high-quality cognitive games instead of a bloated game catalog.

## Stack

- Vite
- React 19
- TypeScript
- React Router
- Plain CSS with a custom visual system

## Starting structure

- Home screen with product framing and the training model
- Game catalog data model for memory, attention, speed, reasoning, spatial, and language
- Adaptive difficulty helper as a placeholder for real tuning logic
- Progress and roadmap sections to guide the next implementation pass

## Scripts

- `bun install`
- `bun run dev`
- `bun run build`
- `bun run lint`
- `bun run typecheck`
